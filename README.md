# Compliant Manager

## Opis projektu
Celem aplikacji „Compliant Manager”, jest usprawnienie procesu reklamacji i zwrotów oraz zapewnienie efektywnego przechowywania informacji o bieżących i archiwalnych zgłoszeniach.

Możliwe jest zarówno korzystanie z interfejsu użytkownika oferowanego przez aplikację, jak i zintegrowanie jej z zewnętrznym systemem poprzez API.

## Tech stack
- .NET 8 (ASP.NET + Entity Framework + Identity Framework)
- Blazor

## Mój wkład
- Frontend aplikacji
- Moduł autentykacji i autoryzacji
- Warstwa serwisów (częściowo)
